What is a Website

Website is a collection of web pages and contents with a domain name and hosted by a web server

Website serve many functions

It can provide information, services, entertainment and a platform for interaction

websites can be accessed through the internet by entering URL into the browser

A typical website has some essential elements

These elements include the following

    Home page which is the front page of the website Navigation, which is the links that users utilize to move between pages Content which are the text, images, videos, and others Functionalities like search features, buttons and others

There are three parts in a website

They include the following

    Content Formatting Interaction

Content

Content are anything we can see and read that explains what the website is about

what the website have to offer

how site visitors can take advantage of what the site offers

Formatting

This is the way text and other content are styled

The way they are presented to make them more presentable

This is the way content are presented to make them more accessible and enjoyable

It make the page easier for different devises usage or responsive design

Interaction

This the way users engage with the website

It can be any action the users take that modify their experiences
